# STM32 Project

- `IDE`:[VS Code](https://code.visualstudio.com/) + [PlatformIO](https://platformio.org/)
- `MCU`:STM32F103C8T6 & STM32F407ZGT6

## How to use?

- `Build`:<kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>B</kbd>
- `Upload`:<kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>U</kbd>
- `Debug`:<kbd>F5</kbd>
